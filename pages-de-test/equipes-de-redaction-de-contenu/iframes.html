Test - Iframes avec et sans Shadow DOM

Informations sur le test

Cette page est utilisée pour tester les bookmarklets d'accessibilité RGAA 4.1.2.

Nom du fichier : iframes.html

Description du test : Cette page contient des éléments <iframe> dans le DOM principal et dans des Shadow DOM pour tester la détection des iframes.

Contenu du test

Cette page contient :

  • Des <iframe> dans le DOM principal
  • Des iframes cachées (hidden, display: none, aria-hidden)
  • Des iframes dans des Shadow DOM
  • Des iframes dans des Shadow DOM imbriqués
