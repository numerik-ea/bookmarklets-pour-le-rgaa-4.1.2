Test - Tableaux avec et sans Shadow DOM

Informations sur le test

Cette page est utilisée pour tester les bookmarklets d'accessibilité RGAA 4.1.2.

Nom du fichier : table-elements.html

Description du test : Cette page contient des éléments de tableau (<table>, <thead>, <tbody>, <tfoot>, <tr>, <th>, <td>) dans le DOM principal et dans des Shadow DOM pour tester la détection des tableaux.

Contenu du test

Cette page contient :

  • Un tableau complet avec <thead>, <tbody>, <tfoot>
  • Un tableau simple sans <thead> / <tbody> / <tfoot>
  • Un tableau de mise en forme (sans en-têtes)
  • Des tableaux dans des Shadow DOM
  • Des tableaux dans des Shadow DOM imbriqués
