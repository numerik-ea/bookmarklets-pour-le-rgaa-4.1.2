Test - Attributs title avec et sans Shadow DOM

Informations sur le test

Cette page est utilisée pour tester les bookmarklets d'accessibilité RGAA 4.1.2.

Nom du fichier : attributs-title.html

Description du test : Cette page contient des éléments avec des attributs title dans le DOM principal et dans des Shadow DOM pour tester la détection des éléments avec attributs title.

Contenu du test

Cette page contient :

  • Des éléments avec et sans attributs title dans le DOM principal (span, lien, image, SVG, bouton, abbr, input ; cas mixtes lien+image)
  • Des Shadow DOM avec title sur l'hôte et/ou le contenu (hôte avec + contenu avec, hôte sans + contenu sans, mixte, hôte sans + contenu avec)
  • Des Shadow DOM imbriqués : niveau 1 avec / niveau 2 sans, et niveau 1 sans / niveau 2 avec
