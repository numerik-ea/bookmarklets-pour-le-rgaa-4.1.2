Test 1.X - Lister les images avec un alt non vide

Informations sur le test

Cette page est utilisée pour tester les bookmarklets d'accessibilité RGAA 4.1.2.

Nom du fichier : 1.X-test.html

Description du test : Cette page contient des images avec différents états d'attribut alt dans le DOM principal pour tester les critères d'accessibilité RGAA 4.1.2 concernant la détection des images avec un attribut alt non vide.

Contenu du test

Cette page contient :

  • Des images avec un attribut alt rempli (doivent être détectées)
  • Des images sans attribut alt (ne doivent PAS être détectées)
  • Des images avec un attribut alt vide (ne doivent PAS être détectées)

Images avec alt rempli dans le DOM principal

Test avec image ayant un alt rempli :

Image avec alt="Image de test avec alt rempli" (doit être détectée)

Test avec image ayant un alt rempli différent :

Image avec alt="Description détaillée de l'image" (doit être détectée)

Test avec image ayant un alt court :

Image avec alt="Logo" (doit être détectée)

Images sans alt ou avec alt vide (ne doivent PAS être détectées)

Test avec image sans attribut alt :

Image sans attribut alt (ne doit PAS être détectée)

Test avec image ayant un alt vide :

Image avec alt="" (ne doit PAS être détectée)
