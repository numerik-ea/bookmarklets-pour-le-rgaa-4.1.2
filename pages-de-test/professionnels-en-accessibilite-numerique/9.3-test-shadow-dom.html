Test 9.3 - Identifier les listes avec Shadow DOM

Informations sur le test

Cette page est utilisée pour tester les bookmarklets d'accessibilité RGAA 4.1.2.

Nom du fichier : 9.3-test-shadow-dom.html

Description du test : Cette page contient des listes (ul et ol) dans le DOM principal et dans des Shadow DOM pour tester les critères d'accessibilité RGAA 4.1.2 concernant l'identification des listes.

Contenu du test

Cette page contient :

  • Des listes <ul> et <ol> dans le DOM principal
  • Des listes dans des Shadow DOM
  • Des listes dans des Shadow DOM imbriqués

  • Deuxième élément de liste non ordonnée
  • Troisième élément de liste non ordonnée
  1. Premier élément de liste ordonnée
  2. Deuxième élément de liste ordonnée
  3. Troisième élément de liste ordonnée
  • Lien de navigation 1
