Test 10.1 - Test présence balises et attributs de présentation avec Shadow DOM

Informations sur le test

Cette page est utilisée pour tester les bookmarklets d'accessibilité RGAA 4.1.2.

Nom du fichier : 10.1-test-shadow-dom.html

Description du test : Cette page contient des balises et attributs de présentation obsolètes dans le DOM principal et dans des Shadow DOM pour tester les critères d'accessibilité RGAA 4.1.2 concernant les balises et attributs de présentation de l'information.

Contenu du test

Cette page contient :

  • Des balises obsolètes (<center>, <font>, <big>, etc.) dans le DOM principal
  • Des attributs obsolètes (align, bgcolor, color, etc.) dans le DOM principal
  • Des attributs width/height invalides dans le DOM principal
  • Des attributs size invalides dans le DOM principal
  • Des éléments similaires dans des Shadow DOM
