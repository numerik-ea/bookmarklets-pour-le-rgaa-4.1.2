Test 7.1 - Lister les liens sans href avec Shadow DOM

Informations sur le test

Cette page est utilisée pour tester les bookmarklets d'accessibilité RGAA 4.1.2.

Nom du fichier : 7.1-test-shadow-dom.html

Description du test : Cette page contient des ancres (<a>) sans attribut href dans le DOM principal et dans des Shadow DOM pour tester les critères d'accessibilité RGAA 4.1.2 concernant les liens sans attribut href.

Contenu du test

Cette page contient :

  • Des ancres sans attribut href dans le DOM principal
  • Des ancres sans attribut href dans des Shadow DOM
  • Des ancres sans attribut href dans des Shadow DOM imbriqués
