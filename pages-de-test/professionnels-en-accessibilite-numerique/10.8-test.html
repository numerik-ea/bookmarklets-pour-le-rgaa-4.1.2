Test 10.8 - Élements cachés

Informations sur le test

Cette page est utilisée pour tester les bookmarklets d'accessibilité RGAA 4.1.2.

Nom du fichier : 10.8-test.html

Description du test : Cette page contient des éléments cachés pour tester les critères d'accessibilité RGAA 4.1.2 concernant les éléments cachés.

Contenu du test

Cette page contient :

  • Des éléments avec aria-hidden="true" (doivent être détectés et mis en évidence avec bordure jaune/bleue et fond rouge)
  • Des éléments cachés avec l'attribut hidden (doivent être détectés)
  • Des éléments cachés avec display: none (doivent être détectés)
  • Des éléments cachés avec visibility: hidden (doivent être détectés)
  • Des éléments visibles (ne doivent PAS être détectés)
