Test 1.X - Vérifier images non contenues dans un lien

Informations sur le test

Cette page est utilisée pour tester les bookmarklets d'accessibilité RGAA 4.1.2.

Nom du fichier : 1.X-verifier-images-non-contenues-dans-un-lien-test.html

Description du test : Cette page contient des images dans différents contextes pour tester les critères d'accessibilité RGAA 4.1.2 concernant la détection des images qui ne sont pas contenues dans un lien.

Contenu du test

Cette page contient :

  • Des images NON contenues dans un lien (doivent être détectées et mises en évidence)
  • Des images contenues dans un lien (ne doivent PAS être détectées)
  • Différents types d'images : <img>, <input type="image">, <area>, éléments avec role="img"
